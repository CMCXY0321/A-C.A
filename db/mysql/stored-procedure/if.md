#### IF
```sql
IF expression THEN 
   statements;
END IF;
```

#### IF ELSE
```sql
IF expression THEN
  statements;
ELSE
  else-statements;
END IF;
```

#### IF ELSEIF ELSE
```sql
IF expression THEN
   statements;
ELSEIF elseif-expression THEN
   elseif-statements;
ELSE
   else-statements;
END IF;
```
